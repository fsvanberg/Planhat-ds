# Colors

All color tokens are CSS custom properties defined in `index.css`. The semantic layer maps to the
**neutral scale** (foundation grays) plus a single **destructive** red. Toggle light/dark mode in
the toolbar to preview both themes live.

---

## Neutral Scale

The primitive gray palette that underpins every semantic token. Based on the Tailwind **neutral**
scale in oklch (chroma = 0).

  <ColorItem
    title="Neutral"

### Semantic → Neutral mapping

| Token | Light | Dark |
|---|---|---|
| `--background` | neutral-50 | neutral-950 |
| `--foreground` | neutral-950 | neutral-50 |
| `--card` | white | neutral-900 |
| `--primary` | neutral-900 | neutral-200 |
| `--secondary` | neutral-100 | neutral-800 |
| `--muted` | neutral-100 | neutral-800 |
| `--muted-foreground` | neutral-500 | neutral-400 |
| `--accent` | neutral-100 | neutral-800 |
| `--border` | neutral-200 | white/10% |
| `--input` | neutral-200 | white/15% |
| `--ring` | neutral-400 | neutral-500 |
