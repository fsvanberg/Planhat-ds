# Planhat Design System

A component library built on [shadcn/ui](https://ui.shadcn.com), [Base UI](https://base-ui.com), and [Tailwind CSS v4](https://tailwindcss.com). All components are copy-owned source — no runtime dependency on a third-party design system package.

---

## Usage

All components are imported directly from `@/components/ui`:

```tsx
import { Button } from '@/components/ui/button'
import { Badge } from '@/components/ui/badge'
import { AlertDialog, AlertDialogTrigger, AlertDialogContent } from '@/components/ui/alert-dialog'
```

Styles live in `src/index.css`. Tokens are CSS custom properties — override `--radius`, `--primary`, etc. to theme the entire system.
